Instação VirtualBox e Vagrant

0 - Execute o script de limpeza (arquivo script_limpeza.sh).
    Baixe o arquivo script_limpeza.sh
    Abrir o terminal (Ctrl+Alt+T), execute os seguintes comandos:
        $ cd ./Downloads
        $ sudo chmod +x script_limpeza.sh
        $ ./script_limpeza.sh
    Aguarde a finalização e passe para o próximo passo

1 - Instalação da VirtualBox
    Atualize os repositórios e instale o VirtualBox (provedor de virtualização):
        $ sudo apt update && sudo apt upgrade -y
        $ sudo apt install virtualbox virtualbox-dkms libelf-dev -y
        
2 - Instalação da Vagrant
    Adicione o repositório oficial do HashiCorp para garantir a versão mais recente:
        $ curl -fsSL https://apt.releases.hashicorp.com/gpg | sudo gpg --dearmor -o /usr/share/keyrings/hashicorp-archive-keyring.gpg
        $ echo "deb [arch=amd64 signed-by=/usr/share/keyrings/hashicorp-archive-keyring.gpg] https://apt.releases.hashicorp.com $(lsb_release -cs) main" | sudo tee /etc/apt/sources.list.d/hashicorp.list

    Edite o arquivo do repositório:
        $ sudo nano /etc/apt/sources.list.d/hashicorp.list
    
    Substitua xia por uma versão suportada, como jammy (Ubuntu 22.04) 
        deb [signed-by=/usr/share/keyrings/hashicorp-archive-keyring.gpg] https://apt.releases.hashicorp.com jammy main

    Atualize os pacotes e instale o Vagrant:
        sudo apt update && sudo apt install vagrant -y
  
    Verifique a instalação:
        virtualbox --version (vai abrir o virtual box). Antes de continuar, feche o virtual box.
        vagrant --version

3 - Vá para o arquivo tutorial_configuração.txt
